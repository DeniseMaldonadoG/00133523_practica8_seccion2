¿Cuál es la diferencia entre autenticación y autorizacion?
Autenticación, Es el proceso de verificar la identidad del usuario, es decir, comprobar que realmente es quien dice ser.

Autorización, Ocurre después de la autenticación. Determina qué permisos o recursos puede usar ese usuario dentro del sistema.

¿Cuál es la función del token JWT en la guía?
es un token de autenticación que el servidor genera después de validar las credenciales del usuario.
